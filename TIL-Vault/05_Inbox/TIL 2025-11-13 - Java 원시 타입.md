## ☕ 자바의 원시 타입 (Primitive Types)과 크기

자바의 원시 타입은 총 8가지가 있으며, 모두 **고정된 크기**를 가집니다.

| **구분**  | **타입 (Type)** | **크기 (Bytes)** | **크기 (Bits)** | **값의 범위 (근사치)**                         |
| ------- | ------------- | -------------- | ------------- | --------------------------------------- |
| **정수형** | `byte`        | **1**          | 8             | $-128$ ~ $127$                          |
|         | `short`       | **2**          | 16            | $-32,768$ ~ $32,767$                    |
|         | `int`         | **4**          | 32            | 약 $\pm 2$억                              |
|         | `long`        | **8**          | 64            | 약 $\pm 9 \times 10^{18}$                |
| **실수형** | `float`       | **4**          | 32            | 소수점 7자리 정밀도                             |
|         | `double`      | **8**          | 64            | 소수점 15자리 정밀도                            |
| **문자형** | `char`        | **2**          | 16            | 유니코드 문자 하나 (0 ~ 65,535)                 |
| **논리형** | `boolean`     | **1**          | 8             | `true` 또는 `false` (JVM 구현에 따라 달라질 수 있음) |
